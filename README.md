# bowling score board

Bowling-score board web-component built with vue.js

Demo: https://kaera.github.io

## Usage

```
<script src="https://unpkg.com/vue"></script>
<script src="https://kaera.github.io/v-game.js"></script>

<v-game></v-game>
```
